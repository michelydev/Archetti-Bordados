# 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de criar uma presença digital moderna para a **Archetti Bordados**, apresentando seus serviços, trabalhos realizados e canais de contato de forma clara e intuitiva.

Além da parte visual, o foco foi proporcionar uma navegação fluida, responsiva e agradável em diferentes dispositivos.

---

# ✨ Funcionalidades

✔️ Página inicial moderna

✔️ Seção institucional

✔️ Catálogo de serviços

✔️ Galeria de trabalhos

✔️ Botão de contato via WhatsApp

✔️ Layout totalmente responsivo

✔️ Navegação intuitiva

✔️ Interface otimizada para desktop e mobile
